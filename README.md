# Pelli.com

Pelli.com is a Telugu matrimony platform designed to help individuals find meaningful connections based on shared values, trust, and compatibility.  
The project features a modern interface, secure authentication, and seamless user experience.


- User Registration and Login with JWT Authentication  
- Profile Creation with Image Upload  
- View and Filter Other User Profiles  
- WhatsApp Chat Integration  
- Forgot Password (OTP Verification)  
- Fully Responsive UI  

---

## Tech Stack

Frontend: HTML, CSS, JavaScript  
Backend:Node.js, Express.js  
Database: MongoDB  
Hosting: Localhost (can be deployed to Render/Heroku in future)
